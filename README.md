# Order-Matching-Engine

An electronic system that matches buy and sell orders for the stock market. Traders can submit several types of orders (List Orders, Market Orders, Immediate Or Cancel Orders & Fill or Kill Orders) to the matching engine which will store them in the order book and then process them.
